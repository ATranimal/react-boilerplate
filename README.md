# React Boilerplate

This is a simple boilerplate that contains the following libraries / functions

- Typescript
- React Router
- Sass
- Storybook
- React Testing Library
- Docker & nginx

If using VSCode, please install Prettier, and add

```
"eslint.validate": [
    "javascript",
    "javascriptreact",
    "typescript",
    "typescriptreact",
]
```

to your settings.json
